## App Title
**Translation App**

**Description:** A multilingual translation application that allows users to input text and get translations into multiple languages using OpenAI's API.

**Structure:**
```
src/
├── app/
│   ├── api/
│   │   └── translate/
│   │       └── route.ts
│   └── page.tsx
├── components/
│   ├── ui/
│   │   ├── TextInput.tsx
│   │   ├── LanguageSelector.tsx
│   │   ├── TranslationModal.tsx
│   │   └── LoadingSpinner.tsx
├── types/
│   └── index.d.ts
└── utils/
    └── translate.ts
```

**Technology:**
- Next.js
- TypeScript
- OpenAI API
